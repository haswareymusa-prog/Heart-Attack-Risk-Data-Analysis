# Basic Heart Attack Risk Data Analysis

This beginner-level Python project explores a heart attack risk dataset containing 8,763 patient records.

## What I Analyzed

- Number of patients and variables
- Patient age range and average
- Cholesterol range and average
- Number and percentage of patients in each heart attack risk group
- Bar graph comparing the risk categories

## Tools Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Main Findings

The patients ranged from 18 to 90 years old, with an average age of approximately 54 years. The average cholesterol level was approximately 260. About 64% of patients were labeled without heart attack risk, while approximately 36% were labeled with heart attack risk.

## Dataset Source

The dataset used in this project is the [Heart Attack Risk Prediction Dataset on Kaggle](https://www.kaggle.com/datasets/iamsouravbanerjee/heart-attack-prediction-dataset).

The dataset file is not included in this repository. Download it from Kaggle and place it in the same folder as the Jupyter notebook before running the analysis.
